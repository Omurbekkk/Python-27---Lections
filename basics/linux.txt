К любой папке можно дописать --help  и выйдет описание как использовать эту команду
cd test     - переходим в папку test (если это папка и она существует)
cd ..       - переход в родительскую папку
cd -        - переход в путь в котором были до этого на 1
cd .        - переход в эту же папку (мы никуда не перейдем)
cd          - переход в домашнюю папку (~)
cd ~        - переход в домашнюю папку (~)
pwd         - показывает абсолютный путь до текущей папки
mkdir test  - создание новой папки test
mkdir test/new dir    - создание новой папки new dir в уже существующей папке test
touch test.txt     - создание файла test.txt
rm test.txt    - удаление файла test.txt
rm -rf test     - удаление папки test со всем ее содержимым
cat test.txt   - просмотр содержимого файла test.txt (не заходя в саму папку. Писать в папке в которой находяться файл)
ls test        - просмотр содержимого папки test
ls -a test     - просмотр всех файлов и папок внутри папки test включая скрытые
ls -l         - просмотр всех файлов и папок с более подробной информацией

nano test.txt    - открыть файл в редакторе nano
Чтобы сохранить (cntr S)
Чтобы выйти (cntr X)
